JAVASCRIPT TIMELINE

  • 1995 - 1996
  • 1996 - 1999
  • 2000 - 2005
  • 2006 - 2008
  • 2009 - 2014
  • 2015

  • 1995 - 1996: THE BIRTH OF JAVASCRIPT

    In 1995, Brendan Eich from Netscape created JavaScript as a scripting language for the Netscape Navigator browser.

    The idea was to allow developers to create more interactive and dynamic web pages.

    JavaScript was originally called Mocha and then LiveScript before finally adopting the name JavaScript.

  • 1996 - 1999: THE STANDARDIZATION OF JAVASCRIPT

    In 1996, JavaScript was standardized for the first time by the European Computer Manufacturers Association (ECMA) and was called ECMAScript.

    ECMAScript 1 was released in 1997, followed by ECMAScript 2 in 1998.

    In 1999, ECMAScript 3 was released, which became the most widely used version of JavaScript for many years.

  • 2000 - 2005: THE RISE OF AJAX AND JAVASCRIPT LIBRARIES

    In the early 2000s, AJAX was developed, allowing web pages to update content without reloading the entire page.

    This led to a greater demand for JavaScript libraries, such as jQuery, which simplified writing JavaScript code.

  • 2006 - 2008: THE BIRTH OF NODE.JS AND ECMASCRIPT

    In 2006, Ryan Dahl created Node.js, allowing developers to use JavaScript on the server side.

    In 2008, ECMAScript 5 was released, which added new features to JavaScript, such as array methods and native JSON.

  • 2009 - 2014: ECMASCRIPT 6 AND THE RISE OF JAVASCRIPT FRAMEWORKS

    In 2009, work began on ECMAScript 6, which added features like classes and promises.

    In 2010, AngularJS was released, a JavaScript framework developed by Google.

    This led to the rise of other frameworks like React and Vue.js.

  • 2015: ECMASCRIPT 2015 AND BEYOND

    In 2015, ECMAScript 2015 (ES6) was released, adding features like let, const, destructuring, and arrow functions.

    Since then, annual releases of ECMAScript have been launched, further improving the language.

    Today, JavaScript is used almost everywhere, from web applications to mobile and desktop devices.
